# 📌 Internship Tracker

A simple and efficient web application to track and manage internship applications in one place. This tool helps students stay organized by keeping track of applications, deadlines, and statuses.

---

## 🚀 Features

- 📋 Add, edit, and delete internship applications  
- 🏢 Track company name and job role  
- 📅 Manage deadlines  
- 🔍 Search and filter applications  
- 📊 View application progress  
- 🔄 Update status (Applied, Interview, Rejected, Selected)  
- 💾 Store data persistently  

---

**Frontend:**  
- HTML, CSS, JavaScript  

**Backend:**
